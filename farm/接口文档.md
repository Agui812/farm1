# 艾萌农场后台管理系统 API 接口文档

## 基本信息

- **基础URL**: `http://localhost:3000/api`
- **返回格式**: JSON
- **通用响应结构**:
```json
{
  "code": 0,
  "message": "OK",
  "data": {}
}
```
- **code说明**: `0` 成功，`404` 未找到，`500` 服务器错误

## 用户管理 (/user)

### POST /user/login
微信登录（用code换取openid）

**请求体**:
```json
{
  "code": "微信wx.login()返回的临时code"
}
```

**响应示例**:
```json
{
  "code": 0,
  "message": "登录成功",
  "data": {
    "id": 1,
    "openid": "oXXXXX真实的微信openid",
    "nickname": "微信用户",
    "avatar": "",
    "phone": "",
    "session_key": "会话密钥",
    "create_time": "2026-04-21T00:00:00.000Z",
    "update_time": "2026-04-21T00:00:00.000Z"
  }
}
```

**说明**:
- 小程序调用 `wx.login()` 获取临时 code
- 后端用 code 调用微信 `jscode2session` 接口换取真实 openid
- 如果 openid 已存在则返回用户信息，否则自动创建新用户
- 返回数据中包含 `session_key`（用于后续解密用户信息等操作）
- 需要在 `.env` 中配置 `WX_APPID` 和 `WX_APPSECRET`

          "num": 2,
          "total_price": 99.90,
          "total_points": 0
        }
      ]
    }

### GET /order/user/:userId
获取指定用户的订单列表

### GET /order/:id
获取单个订单详情（含订单商品明细）

### POST /order
创建订单

**请求体**:
```json
{
  "user_id": 1,
  "address_id": 1,
  "items": [
    { "product_id": 1, "sku_id": 1, "num": 2 },
    { "product_id": 2, "sku_id": 3, "num": 1 }
  ],
  "freight": 0,
  "delivery_type": 1
}
```

**说明**:
- items 中的 sku_id 为商品规格ID，后端会根据 sku_id 从 product_sku 表获取价格
- 如果 items 中直接传了 price 字段，则优先使用传入的价格
- 自动计算商品总价
- 自动应用会员折扣（从config表读取member_discount）
- 自动生成订单编号
- 自动扣减库存

### PUT /order/:id/status
更新订单状态

**请求体**:
```json
{
  "order_status": 3
}
```

**订单状态码**:
| 状态码 | 说明 |
|--------|------|
| 1 | 待付款 |
| 2 | 待发货 |
| 3 | 待收货 |
| 4 | 已完成 |
| 5 | 已取消 |

**自动时间戳**:
- 状态改为2时：自动设置pay_time
- 状态改为3时：自动设置delivery_time
- 状态改为4时：自动设置finish_time

|------|------|------|
| id | int | 主键 |

## 积分商品管理 (/point-goods)

### GET /point-goods/all/list
获取所有积分商品列表

**响应示例**:
```json
{
  "code": 0,
  "message": "OK",
  "data": [
    {
      "id": 1,
      "name": "农家小米1kg",
      "image": "https://picsum.photos/id/131/400/400",
      "points": 100,
      "stock": 50,
      "status": 1,
      "create_time": "2026-04-23T10:00:00.000Z"
    }
  ]
}
```

### GET /point-goods/:id
获取单个积分商品详情

### POST /point-goods
创建积分商品

**请求体**:
```json
{
  "name": "农家小米1kg",
  "image": "https://picsum.photos/id/131/400/400",
  "points": 100,
  "stock": 50,
  "status": 1
}
```

### PUT /point-goods/:id
更新积分商品

### DELETE /point-goods/:id
删除积分商品

### point_exchange_order (积分兑换订单表)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| order_no | varchar(30) | 订单编号 |
| user_id | int | 用户ID |
| goods_id | int | 商品ID |
| goods_name | varchar(100) | 商品名称快照 |
| goods_image | varchar(255) | 商品图片快照 |
| points | int | 消耗积分 |
| num | int | 兑换数量 |
| address_id | int | 收货地址ID |
| status | tinyint | 状态 1=待发货 2=已发货 3=已完成 4=已取消 |
| create_time | datetime | 创建时间 |

## 积分兑换订单 (/point-exchange)

### GET /point-exchange/all/list
获取所有积分兑换订单

**响应示例**:
```json
{
  "code": 0,
  "message": "OK",
  "data": [
    {
      "id": 1,
      "order_no": "PE1234567890ABC",
      "user_id": 1,
      "user_nickname": "测试用户",
      "user_phone": "13800138000",
      "goods_id": 1,
      "goods_name": "农家小米1kg",
      "goods_image": "https://picsum.photos/id/131/400/400",
      "points": 100,
      "num": 1,
      "status": 1,
      "create_time": "2026-04-23T10:00:00.000Z"
    }
  ]
}
```

### GET /point-exchange/user/:userId
获取指定用户的积分兑换订单

### POST /point-exchange
积分兑换（扣除积分并创建订单）

**请求体**:
```json
{
  "user_id": 1,
  "goods_id": 1,
  "address_id": 1,
  "num": 1
}
```

**说明**:
- 自动扣除用户积分
- 自动扣减商品库存
- 自动创建积分兑换订单
- 校验积分余额和商品库存

**响应示例**:
```json
{
  "code": 0,
  "message": "兑换成功",
  "data": {
    "order_id": 1,
    "order_no": "PE1234567890ABC",
    "points_deducted": 100,
    "remaining_points": 400
  }
}
```

### PUT /point-exchange/:id/status
更新兑换订单状态

**请求体**:
```json
{
  "status": 2
}
```

**状态码**: 1=待发货 2=已发货 3=已完成 4=已取消

## 积分管理 (/user-points)

### GET /user-points/user/:userId
获取指定用户的积分明细列表

**响应示例**:
```json
{
  "code": 0,
  "message": "OK",
  "data": [
    {
      "id": 1,
      "user_id": 1,
      "points": 100,
      "total_points": 100,
      "type": "sign",
      "remark": "每日签到奖励",
      "create_time": "2026-04-23T10:00:00.000Z"
    }
  ]
}
```

### GET /user-points/user/:userId/total
获取指定用户的当前总积分

**响应示例**:
```json
{
  "code": 0,
  "message": "OK",
  "data": {
    "user_id": 1,
    "total_points": 500
  }
}
```

### POST /user-points
积分变动（增加或扣除）

**请求体**:
```json
{
  "user_id": 1,
  "points": 100,
  "type": "sign",
  "remark": "每日签到奖励"
}
```

**说明**:
- points 为正数表示增加，负数表示扣除
- type 可选值：order-订单消费，sign-签到，recharge-充值，exchange-兑换，hd-活动
- 自动校验积分余额，扣除时不足会报错
- 自动计算新的总积分并保存
